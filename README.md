End-to-end food safety risk analysis using Python, SQL, and Power BI
# 🍽️ Food Safety Risk Intelligence System

## 📌 Project Overview

This project analyzes food safety risk using environmental and microbial factors such as temperature, humidity, storage time, pH, and bacterial count.

The system assigns a **Risk Score** and classifies food into:

* Safe
* Monitor
* Unsafe

---

## 🚀 Tools & Technologies

* Python (Pandas)
* SQL (SQLite)
* Power BI (Dashboard Visualization)

---

## 📊 Key Features

* Risk scoring model based on real-world food safety parameters
* Automated classification system (Safe / Monitor / Unsafe)
* Interactive Power BI dashboard
* Data-driven insights for decision making

---

## 📈 Dashboard Insights

* High temperature and bacterial count increase food risk
* Meat, fish, and chicken show highest risk levels
* Fruits show lowest risk levels
* Storage conditions directly impact food safety

---

## 📁 Project Structure

* data → dataset
* python → analysis scripts
* sql → queries
* powerbi → dashboard file
* screenshots → dashboard preview

---

## 🎯 Business Value

This project demonstrates how data analytics can be used to monitor food safety and reduce risk in supply chains.
